# Installation

Use `sh-file` for your system:

```shell
darwin.sh
```

```shell
linux.sh
```

Which installed nix, home-manager and activate this configuration.

## Use configuration

Just make changed in configuration files then activate it:

```shell
$ home-manager switch
```
